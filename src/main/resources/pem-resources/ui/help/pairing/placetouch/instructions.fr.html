Modbus

Ajouter un appareil modbus

Utilisez le panneau "Ajout manuel" ci-dessous.
  1. Précisez un nom pour votre nouvel appareil.
  2. Précisez sa localisation.
  3. Précisez l'adresse où il est joignable.
  4. Sélectionnez son profil.
  5. Cliquez sur "Ajouter" pour ajouter l'appareil au boitier PlaceTouch.
